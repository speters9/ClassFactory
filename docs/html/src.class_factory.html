class src.class_factory.ClassFactory.ClassFactory(lesson_no: int, syllabus_path: str | Path, reading_dir: str | Path, slide_dir: str | Path, llm, project_dir: str | Path | None = None, output_dir: str | Path | None = None, lesson_range: range | None = None, **kwargs)[source]

Bases: object

A factory class responsible for creating and managing different educational modules.

This class provides a unified interface to create instances of three modules:

BeamerBot: Automated LaTeX Beamer slide generation ConceptWeb: Automated concept map generation QuizMaker: Automated quiz generation, hosting, and analysis

create_module(module_name: str, **kwargs) → BeamerBot | ConceptMapBuilder | QuizMaker[source]

Create a specific module instance based on the provided module name.

Parameters:

  • module_name (str) – The name of the module to create. Options are ‘BeamerBot’, ‘ConceptWeb’, or ‘QuizMaker’.

  • **kwargs – Additional keyword arguments for the specific module.

Returns:

The created module instance.

Return type:

Union[BeamerBot, ConceptMapBuilder, QuizMaker]

Raises:

ValueError – If an invalid module name is provided.

By default, each module output will be saved in its respective directory in the ClassFactoryOutput/ directory, in a directory named after the designated lesson number.
